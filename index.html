chronologie des opérations pour créer son site Web sur gitHub

Bonjour, voici comment vous pourrez montrer votre site à qui vous le souhaitez.

  1. inscrivez-vous (si ce n'est déjà fait) sur https://github.com/ (sign up en haut à droite) ;
  2. sur la page "Create your account", proposer un identifiant ("username"), s'il n'est pas déjà pris un symbole de validation en vert apparaît, sinon trouver un identifiant court facile à mémoriser et original (non utilisé) ; ensuite renseigner votre adresse de courriel ;
  3. pour le mot de passe de minimum 15 caractères : ouvrir votre boîte mail sur une autre page , envoyez-vous un mail contenant votre login, en sujet mettre "github", et votre M P tapé au hasard avec des minuscules et des majuscules et des caractères spéciaux comme #{@]^ et des chiffres. Une fois envoyé archiver ce mail dans un libellé (dossier) comme "gestion" ou "comp" ; copier coller votre mp dans la page d'inscription de github.
    Résoudre l'énigme anti-robots et valider ! Bienvenue sur github, le site de partage de code le plus connu dans le monde !
  4. configuration :
    sur la page suivante à "What kind of work do you do, mainly?" cliquer sur student
    à How much programming experience do you have? cliquer sur (ce que vous voulez) A little par exemple
    à What do you plan to use GitHub for? choisir trois options au max
    create a website
    Host a p
    Collab ...
    Exprimez-vous sur vos centres d'intérêts (pro)
    Puis cliquer sur Complete setup (finaliser les réglages)
  5. aller dans votre boîte mail pour confirmer votre inscription : github vous a envoyé un mail dont il faut cliquer le bouton.
    Cela va ouvrir une page d'accueil (que vous pouvez obtenir en français)

    6. Pour créer le site :

    1. Créer un 'repository' ou référentiel par exemple :
      sur la coppie d'écran ci-dessous, je crée un projet ou référentiel 'essai jodenda/ essai
    2. cliquer sur l'option publique et valider ! (creating repository)
      3. image.pngpuis créer un fichier index.html et écrire du code (copier coller une page d'exemple de w3shool ou une de vos pages) Une fois fini TRES IMPORTANT cliquer sur Commit your file pour l'enregistrer, sinon c'est perdu ! image.png Voilà le résultat : image.png Cliquer sur settings (réglages) pour mettre tout ça en ligne ! en bas de la page réglage activer le site image.png image.png image.png C'est fini ! votre page est en ligne mais cela peut prendre un peu de temps avant l'appartition réelle de votre page exemple ici https://jodenda12.github.io/essai/
